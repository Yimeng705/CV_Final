# Poster大纲

## 格式规格
- **版式**：横版（参考CVPR格式）
- **比例**：约4:3（宽:高）
- **推荐尺寸**：120cm × 90cm 或按比例缩放
- **语言**：中文为主，关键术语保留英文

┌──────────────────────────────┐

│    自适应高斯密度控制         │
├──────────────────────────────┤
│                              │
│ 原始3DGS密度控制              │
│ ┌──────────────────────────┐ │
│ │ grad > τ_grad ?          │ │
│ │   ├─ small scale → Clone │ │
│ │   └─ large scale → Split │ │
│ │ opacity < τ_α → Prune   │ │
│ └──────────────────────────┘ │
│ ❌ 问题：仅几何梯度，语义     │
│    边界处阈值过高→欠采样     │
│                              │
│ 本文改进：双驱动阈值          │
│ ┌──────────────────────────┐ │
│ │ ① 语义边界检测:          │ │
│ │  SemScore(i) =           │ │
│ │  mean∥sem(i)-sem(j)∥²   │ │
│ │  j ∈ KNN(i)             │ │
│ │                          │ │
│ │ ② 双驱动阈值调整:         │ │
│ │  τ_adj = τ_grad /        │ │
│ │  (1 + α·SemScore)       │ │
│ │                          │ │
│ │ ③ 语义感知密度判定:      │ │
│ │  grad > τ_adj ?         │ │
│ │   ├─ Clone (语义增强)   │ │
