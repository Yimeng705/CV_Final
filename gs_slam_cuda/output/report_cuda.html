3. SA-AGD 核心创新

3.1 方法描述

双路径密度控制:
路径1 (几何): 投影覆盖度 + 深度加权 → 几何重要性评分
路径2 (语义): GPU KNN (torch.cdist) → 语义边界评分 → 额外clone增强
融合: should_densify() 结合两条路径的评分进行决策

7. 整体架构

四篇论文融合架构:
• 输入: 单目RGB图像序列 (TUM/EuRoC/Replica)
• 前端: MASt3R-SLAM 点图匹配 + 位姿估计
• 建图: 3DGS综述 tile-based渲染 + SA-AGD密度控制
• 语义: OpenMonoGS-SLAM KNN语义边界检测
• 后端: MASt3R-Fusion 层次化因子图优化
• 输出: 渲染图像 + 优化轨迹 + 3D PLY高斯模型

核心结论:
1. SA-AGD 在纯几何密度控制基础上额外增加了 8 个语义边界增强操作
2. Chamfer距离从 0.0415 降至 0.3810 (改善 -818.1%)
3. 高斯数在合理范围内增长 (1.07x)，不会导致显存溢出
4. CUDA加速使渲染从 ~210ms 降至 ~630ms (~0x)
5. RTX 3060 8GB VRAM足以支持 500K+ 高斯的实时渲染
